# rock-paper-scissors

Rock Paper Scissors game for the Odin Project.
Game is played entirely in the console, no GUI.


GUI additions:
- R: three buttons
- R: display results
- P: enlarge button on hover
- P: icon on button
- R: display running score
- R: display winner who got 5 points
